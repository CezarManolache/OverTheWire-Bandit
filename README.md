## ⚠️ Disclaimer: Read Before Proceeding!

Acest repository conține **write-up-uri și metodologii** pentru jocurile OverTheWire. 
Dacă ești aici pentru a învăța, te rog să ții cont de următoarele:

1. **Don't Just Copy-Paste:** Scopul acestor jocuri este să îți dezvolți gândirea critică și abilitățile de Linux. Dacă doar copiezi comenzile, nu vei învăța nimic.
2. **Try First:** Încearcă să rezolvi nivelul singur timp de cel puțin 30-60 de minute înainte de a te uita la soluție.
3. **Understand the "Why":** Nu te uita doar la comandă. Citește secțiunea **"Tech Focus"** sau **"Learning"** pentru a înțelege de ce am folosit acea metodă.
4. **No Credentials:** Conform regulilor OverTheWire, acest repository **NU** conține parolele (flag-urile) finale. Va trebui să rulezi comenzile singur pe server pentru a le obține.

> *"Hacking is not about the destination (the password), it's about the journey (the process)."*
